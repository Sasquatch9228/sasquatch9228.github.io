SWAB (Solitude Without A Browser) - an unblocked Google Search exploit for school-managed Chromebooks, with a read-only file explorer. This exploit is based on Quartinal's Baghdad exploit (which is similar to @Brandon421-ops's Incognito On The Sign-In Screen exploit).

    Instructions

  1. First, sign out of your Chromebook if you have not already.
    2. Connect to the DNS server
  2. Open the toolbar menu by clicking the time in the bottom right.
  3. In the toolbar menu, click and open the Wi-Fi menu.
  4. Click the settings button to open the Wi-Fi settings.
  5. Expand the Network dropdown.
  6. Select Custom name servers.
  7. Set the a minimum of one of the first input field value boxes to the DNS serve address which is 52.207.185.90.
  8. If it does not automatically save your modicications, disconnect from your network 1-3 times.
  9. Then, reconnect to your network 1-3 times.
  10. If a Network not available page appears:

    Click Sign in as an existing user.

    Click the Back button until it shows your login page.

    Under your login page, there should be a Sign in with Google account button. Click it, and then click Forgot email?.

    It should now show a 400 error page. On the 400 error page, there should be a clickable Google logo. Click the logo.

    You should now have an unblocked Google search webpage open. If you do not, feel free to submit an issue on the GitHub repository linked above and Quartinal will try to help resolving the issue.
